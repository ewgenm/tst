# 📗 ТЗ №2: FRONTEND (Vue 3 + TypeScript)
**Проект:** TaskSync Team  
**Стек:** Vue 3 (Composition API), Vite, TypeScript, Pinia, Vue Router, Tailwind CSS, Radix Vue, Axios, Laravel Echo  
**API Endpoint:** `http://tst.test/api/v1`  
**Источник истины:** ТЗ №0 v1.2

## 1. Цель и Архитектура
Создать реактивное SPA с поддержкой тем, drag-and-drop, real-time обновлениями и строгой типизацией. Архитектура: **Components → Composables → Stores (Pinia) → API Layer → Router**.

## 2. Настройка проекта
```bash
npm create vite@latest tasksync-frontend -- --template vue-ts
cd tasksync-frontend
npm install vue-router pinia axios laravel-echo pusher-js @vueuse/core
npm install -D tailwindcss postcss autoprefixer @types/node
npx tailwindcss init -p
```

**Структура папок:**
```
src/
├── api/          # Axios instance, endpoints, types
├── assets/       # Styles, icons
├── components/   # UI Kit, Layout, Features
├── composables/  # useTasks, useTheme, useDragDrop
├── layouts/      # AppLayout, AuthLayout
├── pages/        # Route views
├── stores/       # Pinia stores
├── types/        # TS interfaces (sync with ТЗ №0)
└── utils/        # Formatters, constants
```

## 3. API слой и Типизация
- Создать `src/types/index.ts` с интерфейсами из ТЗ №0 (`User`, `Task`, `Project`, etc.).
- Настроить Axios (`src/api/client.ts`):
  - Base URL: `import.meta.env.VITE_API_URL || 'http://tst.test/api/v1'`
  - Interceptors: attach CSRF token, handle 401/403, unwrap `data` from response.
- Создать репозитории/сервисы: `taskApi.ts`, `projectApi.ts` с типизированными методами.

## 4. Состояние (Pinia)
- `useAuthStore`: user, token, login, logout, fetchMe.
- `useTaskStore`: tasks list, filters, pagination, CRUD actions, optimistic updates.
- `useProjectStore`: projects, currentProject, members.
- `useUIStore`: sidebar state, theme, modals, toasts.
- **Важно:** Использовать `persistedstate` плагин для сохранения темы и фильтров.

## 5. Маршрутизация и Макеты
- `router/index.ts`: lazy loading, guards (`requiresAuth`, `canAccessProject`).
- Layouts:
  - `AppLayout.vue`: Sidebar (collapsible), Header, Main content area, Toast container.
  - `AuthLayout.vue`: Login/Register forms centered.
- Views: `/inbox`, `/projects/:id`, `/today`, `/habits`, `/settings`.

## 6. UI Компоненты и Тема
- **Tailwind Config:** Расширить палитру, настроить `darkMode: 'class'`.
- **Theme Toggle:** 
  - Хранить в `localStorage` и синхронизировать с БД через API.
  - Класс `dark` вешается на `<html>`.
- **UI Kit (Headless):**
  - `Button`, `Input`, `Select`, `Modal`, `Dropdown`, `Tooltip` (Radix Vue).
  - Кастомные стили через Tailwind utilities.
- **Task List:** 
  - Компонент `TaskItem.vue` с чекбоксом, приоритетом, дедлайном, аватаром.
  - Drag & Drop: `vuedraggable` или `@dnd-kit` (обертка для Vue). Обновлять `position` на сервере debounce.
- **Forms:** `vee-validate` + `zod` для клиентской валидации перед отправкой.

## 7. Real-time и Уведомления
- Настроить `Echo` в `main.ts`:
  ```ts
  import Echo from 'laravel-echo';
  import Pusher from 'pusher-js';
  window.Echo = new Echo({ broadcaster: 'reverb', key: 'tasksync-key', wsHost: 'tst.test', wsPort: 8080 });
  ```
- Подписка в `useTaskStore`: `Echo.private(`project.${projectId}`).listen('TaskUpdated', ...)`.
- Компонент `NotificationCenter.vue`: список unread, mark as read, toast alerts.

## 8. Производительность и UX
- Virtual scroll для длинных списков задач (если >100).
- Skeleton loaders при fetch.
- Debounce поиска и фильтрации.
- Keyboard shortcuts: `N` (new task), `/` (search), `Esc` (close modal).
- Accessibility: `aria-*`, focus trapping в модалках, контрастность тем.

## 9. 🤖 Инструкции для ИИ-агента (Frontend)
1. **Генерируй по слоям:** Types → API Client → Stores → Layout → Pages → Components.
2. **Строго типизируй.** Все пропсы, emits, API responses должны соответствовать ТЗ №0.
3. **Используй Composition API (`<script setup>`)** и Pinia для состояния.
4. **Не используй тяжелые UI-фреймворки.** Только Headless + Tailwind.
5. **Реализуй тему через CSS-переменные и класс `dark` на `<html>`.**
6. **Prompt-ша!блон для ИИ:**
   ```

   Не меняй структуру API запросов.
   ```
